Thẻ có thứ tự(ol):

Quá trình học web:

  1. Nhập môn web và ứng dụng.
  2. Lập trình web.
  3. Xây dựng ứng dụng web.

Thẻ có thứ tự đổi kiểu(type):

Quá trình học web theo ABC:

  2. Lập trình web.
  3. Xây dựng ứng dụng web.
